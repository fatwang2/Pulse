Pulse 0.4.0

What's new:
- Cryptocurrency search, quotes, and charts now use Binance Spot public market-data endpoints, with one-second WebSocket ticker updates while the menu-bar popover is open. No Binance account or API key is required.
- Crypto symbols now use structured base/quote pairs and are displayed consistently as `BTC/USDT`. Existing Yahoo-style symbols such as `BTC-USD` are migrated automatically.
- Binance is now the single cryptocurrency source, avoiding mismatched instruments and inconsistent names between Yahoo and Binance. The current Binance Spot symbol directory is cached locally and refreshed automatically.
- Binance and Longbridge live streams can run together, so crypto and securities continue receiving updates from their appropriate providers.
- Data-source settings now use consistent status labels and concise market summaries, with connection details and complete capabilities available inside each source.
- Pulse now includes a small set of anonymous product-usage events through TelemetryDeck. Analytics are enabled by default and can be disabled at any time in Settings. Pulse never attaches symbols, watchlists, positions, search text, credentials, or other user-provided content to these events; the full collection scope is documented publicly in the README.
- Repeated local Debug builds now reuse the developer's own Apple Development identity when available, reducing unnecessary Keychain authorization prompts without affecting release signing.

For first-time installation, download Pulse-0.4.0.dmg and drag Pulse to Applications. The zip asset is used by Sparkle automatic updates.
